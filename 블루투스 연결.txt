1) 패키지 업데이트 하기

sudo apt-get update

sudo apt-get upgrade

2) 블루투스 패키지 설치하기

sudo apt-get install bluetooth blueman bluez

3) 라즈베리파이 재부팅하기

sudo reboot

4) 블루투스 관련 파이썬 패키지 설치하기

sudo apt-get install python-bluetooth

5) 블루투스 페어링하기

sudo bluetoothctl 


[bluetooth]#power on

[bluetooth]#agent on

[bluetooth]#discoverable on

[bluetooth]#pairable on

[bluetooth]#scan on
